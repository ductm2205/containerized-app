# DEMO SPRING BOOT + DOCKER

This project is a demo for creating a pipeline on Jenkins with AWS

## Prequisites

You need these installed on your system to run this project

    1. Docker
    2. Docker Compose
    3. Java 17
    4. Maven (if you don't like docker)

## Getting started

Follow these:

    1. Create a copy of .env.example named .env, then fill in your configuration
    
        ```
            cp .env.example .env
        ```
    2. Run the project using maven or Docker
